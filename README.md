# FreshMart Premium Supermarket

This is a Vite + React project for the FreshMart Premium Supermarket frontend. This repository was initialized from an existing local project.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

Notes
- The project uses `vite` and `@vitejs/plugin-react`.
- Keep any private keys or credentials out of the repo (use environment variables and the GitHub Actions/hosting provider secret settings).
